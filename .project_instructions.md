# Gavi Reminders — Project Memory

## Overview
Cross-platform React Native (Expo, expo-router) MVP replicating the iOS Reminders app. **100% Spanish UI.** Frontend-only: local persistence via expo-sqlite, NO backend/auth/cloud.

## Tech & Structure
- expo-router file routes under `app/`. Entry: `app/_layout.tsx` (GestureHandlerRootView > SafeAreaProvider > ErrorBoundary > ThemeProvider > DataProvider > Stack).
- Tabs at `app/tabs/` (index=Inicio dashboard, search=Buscar, settings=Ajustes). Other routes: `smart/[type]`, `list/[listId]/{index,edit}`, `list-create`, `reminder/add`, `reminder/[reminderId]/{index,edit}`, `tags`, `tag/[tagName]`.
- Fonts: Nunito via @expo-google-fonts/nunito (loaded in _layout, fallback System/Roboto/Arial in ThemedText).
- Reanimated v4 (+ react-native-worklets/plugin in babel.config.js).
- metro.config.js: added `'wasm'` to assetExts — REQUIRED for expo-sqlite web export to build.

## Data Layer
- `services/database.ts`: SQLite (gavi_reminders.db). Tables List, Reminder, Tag, ReminderTag, Setting. Seeds 3 default lists (🛒 Compras, 🏠 Casa, ✈️ Viajes). All CRUD + smart counts + search here.
- `services/notifications.ts`: expo-notifications scheduling (DATE/DAILY/WEEKLY/MONTHLY triggers), no-op on web.
- `context/DataContext.tsx`: db init, `refreshToken`/`triggerRefresh()`, and high-level action wrappers (saveNewReminder, editReminder, toggleComplete, removeReminder, removeList) that also handle notifications. Screens reload via useFocusEffect keyed on refreshToken.
- `context/ThemeContext.tsx`: light/dark/system theme persisted in Setting table.

## Design
- Theme tokens in `theme/theme.ts` (lightColors/darkColors, SMART_COLORS, LIST_COLORS, EMOJI_OPTIONS, FONTS). Primary #007AFF, accent #FF9500.
- Key components: ThemedText, ScreenHeader, ReminderItem (Swipeable delete on native, inline delete button on web), AnimatedCheckbox, SmartCard, ListRow, Fab (gradient), OptionSheet, ConfirmSheet, SegmentedControl, ListForm, ReminderForm, DateTimeField (native picker / HTML input on web), TagChip, EmptyState, ReminderList.

## Notes
- Navigation version packages pinned in package.json (^7.4.0 / ^7.3.16 / ^7.1.8) to satisfy expo-doctor.
- JSX text children must use literal accented chars (not \u escapes); string-literal props can use escapes.
- All builds/tests pass (expo doctor + web export + dev server).
